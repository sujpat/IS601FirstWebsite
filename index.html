Purpose

The purpose of this page is to practice HTML and using LINUX commands to practice git

Linux Command Examples

		ls = directory listing
		cd = change directory
		pwd = present working directory
		mkdir = make directory
		rm = delete

VI Command Examples

                vi filename = enter file and view code
                i = insert text
                :w = write
                :q = quite
                :wq = write and quit

Git Command Examples

                git status = show file in directory
                git add = add file to commit
                git commit -m = commit the file and add description
                git push origin = push code to host
                git merge = merge branch with main
